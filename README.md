## Pull Request 규칙 ( Optional )

1. 새로운 브랜치를 생성: `git checkout -b feature/week1/meiko`
2. 변경사항을 커밋: `git commit -m 'feature : some awesome features'`
    - README 파일은 `README.md` 로 고정
    - 주차 별 디렉토리 생성
3. 브랜치에 Push: `git push origin feature/week1/meiko`
4. Pull Request를 생성
